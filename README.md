## Техническое задание на разработку программы "Простой калькулятор".

### 1. Введение
Программа представляет собой консольное приложение-калькулятор, с помощью которого пользователь сможет вычислять простейшие арифметические операции.

### 2. Цель
Предоставление пользователю простой и удобный калькулятор, для простейших арифметических вычислений

### 3. Требования к программе
- Быстрые и точные вычисления
- Обработка ошибок
- Выполнение арифметических вычислений с операциями: сложения, вычитания, деления, умножения

### 4. Тест кейсы

| Номер теста | Ввод | Вывод |
| ------------- | ------------- | ------------- |
| 1  | 5 + 10  | Результат: 15 |
| 2 | 10 - 5  | Результат: 5 |
| 3 | Ввод символа вместо цифры  | Ошибка! Введите число |
| 4 | Неправильный ввод операции  | Ошибка! Используйте только: + - * / |
| 5 | Деление на 0  | Ошибка! На ноль делить нельзя! |
